# 頒獎典禮座位預約系統

供學校（教務處／學務處／輔導室）在頒獎典禮前預約視聽教室座位的單頁網頁工具，無需後端即可使用。

## 功能

- **座位配置**：視聽教室 4＋7＋4 × 15 排，共 225 席
- **三處室分色預約**：教務處、學務處、輔導室各自獨立的顏色與統計
- **多場次管理**：依典禮日期切換座位表，可提前預約未來場次
- **點選／拖曳框選**：單擊預約一席，拖曳一次框選多席；支援觸控
- **框選刪除／全部刪除**：批次解除本處室預約
- **匯入名單**：下載 Excel 模板填寫後上傳（.xlsx / .csv），匯入前自動檢查格式、座位範圍與衝突
- **匯出 CSV**：含日期、排、區、座號、處室、獎項、姓名
- **列印**：列印時自動隱藏工具列，只留座位圖

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，資料儲存在瀏覽器 localStorage（單機模式）。

### 多台電腦同步（選用）

預設為單機模式。若需多台電腦即時同步：

1. 到 [Firebase Console](https://console.firebase.google.com) 建立專案
2. 新增 Realtime Database
3. 專案設定 → 一般 → 新增網頁應用程式，取得 `firebaseConfig`
4. 將 `index.html` 內的 `const FIREBASE_CONFIG = null;` 改為貼上你的設定

頁面右上角會顯示「雲端同步中」表示連線成功。
